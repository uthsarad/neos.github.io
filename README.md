# neos.github.io

NeOS website with multi-section landing experience (Home, Download, News, Support, Help, About).

## Theme behavior

- Site adapts automatically to local device time using 5 themes: dawn, morning, afternoon, dusk, night.
- Manual theme toggle is available in the footer and persists in local storage.
- Fallback defaults align with OS preference: morning for light mode, night for dark mode.

## Run locally

- Open `index.html` directly, or
- Serve with `python3 -m http.server 4173` and visit `http://127.0.0.1:4173`.
